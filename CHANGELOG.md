# 变更日志

本文件记录项目的重要变化。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增

- 待办清单已实现一层可选子任务，可用 Tab / Shift+Tab 调整层级，并提供父任务行上的可发现鼠标入口；该功能尚未完成打包应用验收或发布。

### 修复

- 待办事项现在会根据便签宽度完整换行并自动调整高度，长网址和连续字符不再被截断；中文输入法确认候选词时也不会误创建下一条待办。

### 变更

- 托盘现在会显示当前版本，点击即可随时查看本版新增与修复。

## [0.1.13] - 2026-07-15

### 变更

- Windows 自动更新改用同一公开更新仓库的 GitHub provider，恢复按版本读取 blockmap 的差分下载。

## [0.1.12] - 2026-07-14

### 修复

- 已命名便签现在会在顶部常驻显示名称，并增强字重与悬停反馈；名称保存失败时不再丢失待保存内容。

## [0.1.11] - 2026-07-12

### 新增

- Windows 自动更新下载期间显示独立、非阻塞的进度窗口；没有百分比时显示明确的进行中状态。
- 支持在便签顶部双击命名、本地保存每张便签的名称，并用名称区分系统窗口。

### 修复

- 修复 Windows 关闭最后一张便签后应用退出的问题，托盘现在会继续常驻，更新完成提示也不会被意外跳过。
- Windows 现在限制为单实例运行；关闭便签后可从托盘“显示所有便签”，再次启动应用也会恢复已关闭的便签窗口。

### 安全

- 升级开发、测试和打包工具链，修复 Vite、Vitest、esbuild 与 tar 的已知依赖漏洞。

## [0.1.10] - 2026-07-11

### 新增

- 建立公开仓库、贡献指南、Issue 模板和持续集成检查。
- macOS 打包版支持启动时静默检查更新，也可以从托盘手动检查。
- 发现新版后可在应用内下载并校验 DMG，下载完成后保存便签、打开安装镜像并退出。

### 变更

- 将公开文档和 GitHub 协作界面统一为中文。
- Mac 更新采用半自动安装流程，用户仍需把应用拖到 Applications 并确认替换。

### 修复

- 修复 Mac 打包版因 `electron-updater` 模块格式不兼容而无法启动的问题。

## [0.1.9] - 2026-07-11

### 新增

- 支持创建多张始终置顶并保存在本地的便签。
- 支持纯文本、可编辑待办清单、便签颜色和透明度。
- 支持粘贴剪贴板图片与系统截图、拖入图片和删除图片。
- 托盘菜单支持新建便签、开机启动、检查更新和退出。
- Windows 打包版通过 `electron-updater` 和 GitHub 通用更新源检查更新。
- 兼容既有本地 profile，但源码和公开安装包不包含任何 profile 数据。
- 提供 Windows x64 NSIS Setup 和未签名 Apple Silicon macOS 构建命令。

### 变更

- Windows 发行方式收敛为一个中性 Setup 安装包，不再构建 portable。
- 长内容在便签内部滚动，工具栏保持可操作。

### 修复

- 退出应用或重启安装更新前，会先保存尚未落盘的便签数据。
- 删除图片后恢复原有编辑焦点，不再调用原生模态确认框。
- 恢复窗口时会把位置和尺寸限制在可用显示器范围内。

## 公开前内部迭代

> `0.1.0` 至 `0.1.8` 是项目公开前的内部迭代版本，没有对应的公开 Tag、Release 或下载资源。以下仅保留经过隐私清洗的用户可见变化。

## 0.1.8 - 2026-07-06

- 修复 Windows 桌面和开始菜单快捷方式仍显示默认 Electron 图标的问题。
- 快捷方式直接引用随应用安装的 `.ico` 文件，降低 Shell 图标解析与缓存影响。
- 增加未签名的 Apple Silicon macOS DMG 构建能力。
- 汇总保留开机启动、长内容滚动和 Windows 图标资源修复。

## 0.1.7 - 2026-07-06

- 首次运行时默认开启开机启动，并写入一次性状态标记。
- 用户关闭开机启动后，后续启动不会再次强制开启。

## 0.1.6 - 2026-07-06

- 恢复 electron-builder 对 Windows exe 图标和版本资源的写入。
- 安装时刷新已有桌面和开始菜单快捷方式。

## 0.1.5 - 2026-07-06

- 接入应用、安装器、卸载器、任务栏和托盘图标资源。
- 暴露出 exe 资源编辑被关闭后，自定义图标不会真正写入的问题。

## 0.1.4 - 2026-07-06

- 增加多配置 Windows 构建能力，验证共享应用身份与 userData 的覆盖升级行为。
- 为后续将程序功能与本地文案解耦积累构建经验。

## 0.1.3 - 2026-07-06

- 修复升级安装时误报应用无法关闭的问题。
- 绕过旧卸载器的错误关闭检测，改为覆盖安装并重新写入卸载登记。
- 保持 `%APPDATA%\floating-sticky-notes` 数据目录不变。

## 0.1.2 - 2026-07-06

- 用于复现和定位 Windows 升级安装器错误提示应用无法关闭的问题。
- 确认问题来自旧卸载器的关闭检测，而不是残留应用进程或安装目录文件锁。

## 0.1.1 - 2026-07-06

- 修复待办事项较多时，小窗口底部内容无法访问的问题。
- 正文、清单和图片共用便签内部滚动区，顶部拖拽栏与工具栏保持固定。
- 保持便签可以使用较小尺寸，不要求用户为了查看内容而放大窗口。

## 0.1.0 - 2026-07-06

- 首次生成可安装的 Windows x64 版本。
- 建立多便签窗口、正文保存、窗口位置与尺寸恢复和 JSON 本地存储。
- 提供待办清单、图片、颜色和透明度等核心功能。
- 开始接入系统托盘与开机启动能力。
